# Purpose

This repository contains my personal analysis for statistical distributions, and several (relevant) solutions from the main textbook reference.

Along the time, this repository wil be updated with my own annotation, analysis and "deviations", one can find them on the specialized Jupyter notebooks on:

1. Exponential distributions: [[./notebooks/01_exponential.ipynb]]
2. Weibull distributions: [[./notebooks/02_weibull.ipynb]]
3. Gamma distributions: [[./notebooks/03_gamma.ipynb]]
4. Normal distributions: [[./notebooks/04_normal.ipynb]]
5. Lognormal distributions: [[./notebooks/05_lognormal.ipynb]]
6. Extreme Value distributions: [[./notebooks/06_extreme_value.ipynb]]
7. Nonparametric distributions: [[./notebooks/07_nonparametric.ipynb]]

It is noteworthy to remark that not all notebook are filled or ready to be evaluated. This sequence is based, mainly, on the sequence presented by Modarres (2016) in his textbook.
